# WURCS-Substitution
The tool to replace subsituents in WURCS. Its a commandline tool (no GUI). That takes:

-f : File with one or multiple WURCS sequences (line by line). There can be empty lines front, back and in between.

-s : WURCS notation of the substituent to replace

-r : WURCS notation of the replacement substituent

-o : New file that will be generated
